KNN (K Nearest Neighbor)#

  1. Pengertian KNN (K Nearest Neighbor)

1.1 KNN merupakan sebuah metode pengelompokan terhadap sekumpulan data berdasarkan pembelajaran data yang sudah terklasifikasi sebelumnya

1.2 Dekat atau jauhnya tetangga biasanya dihitung berdasarkan Euclidean Distance.

1.3 Rumus Euclidean Distance adalah sebagai berikut:

\[D(x,y) = \sqrt(\Sigma_I^n(xi-yi)^2)\]

1.4 dimana D(a,b) adalah jarak skalar dari dua buah vektor data a dan b yang berupa matrik berukuran d dimensi

1.5 berikut adalah algoritma KNN:

  • Menentukan parameter k (jumlah tetangga paling dekat)

  • menghitung kuadrat jarak euclidean objek terhadap data training yang diberikan

  • mengurutkan secara ascending

  • mengumpulkan kategori Y (Klasifikasi nearest neighbor berdasarkan nilai k)

  1. Perhitungan Manual

Berikut merupakan perhitungan manual menggunakan euclidean distance pada excel, terdapat 5 missing values pada sepal length dengan setosa sebagai variety nya

2.1 Terdapat 50 data yang terdiri dari id, sepal.length, sepal.width, petal.length, petal.width, dan variety.

2.2 Pisahkan baris data yang terdapat missing values. Pada data berikut terdapat 5 missing values yang terletak pada kolom sepal.length dalam varietas setosa, kemudian hitung rata-rata pada masing-masing kolom sepal width, petal.length, dan petal.width.

2.3 Hitung jarak menggunakan rumus euclidean distance.

2.4 Urutkan untuk mengetahui Nearest Neighbor dengan rumus sebagai berikut:

=RANK.AVG(N10;\(N\)10:\(N\)54;1)

2.5 Nilai null kemudian diisi menggunakan nilai yang sering muncul pada K=5

Teori#

Deteksi outlier adalah proses mengidentifikasi dan menangani data yang tidak biasa atau tidak terduga dalam sebuah kumpulan data.

rumus manhattan distance:

\[d(x,y) = \sqrt{\sum_{i=1}^{n}(x_i - y_i)}\]

Perhitungan LOF secara Manual#

Berikut merupakan perhitungan LOF dengan 3 dimensi dan jumlah data sebanyak 10 data. K = 2

1.1 Mencari distance dengan menggunakan rumus manhattan distance, lalu diurutkan. berikut adalah rumus yang digunakan dalam excel:

  • manhattan distance: =ABS(\(B\)2-B3)+ABS(\(C\)2-C3)+ABS(\(D\)2-D3)

  • pengurutan: =RANK.AVG(H2;\(H\)2:\(H\)10;1)

1.2 Mencari N dan jumlah objek yang memiliki nilai kurang dari sama dengan K.

1.3 Mencari LRD: dengan jumlah N dibagi dengan sigma reachdist

1.4 Mencari LOF dengan rumus excel sebagai berikut: =(L26+L28)*(56+54)

1.5 Melakukan pengurutan: =RANK.AVG(O25;\(O\)25:\(O\)34;1)

Setelah pengurutan, maka dapat disimpulkan bahwa Outlier nya adalah a.

import numpy as np
import pandas as pd
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import GaussianNB
from sklearn.metrics import accuracy_score
import csv

# Load the iris dataset
iris = load_iris()
X = iris.data
y = iris.target

# Initialize the Naive Bayes classifier
nb_classifier = GaussianNB()

# Split the data into training and test sets (50% training, 50% test)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.5, random_state=0)

# Train the classifier
nb_classifier.fit(X_train, y_train)

# Predict the classes for the training set
y_train_pred = nb_classifier.predict(X_train)

# Combine features and target into one list
train_data = np.column_stack((X_train, y_train))

# Write the training data to a CSV file
with open('training_data.csv', mode='w', newline='') as file:
    writer = csv.writer(file)
    # Write header
    writer.writerow(iris.feature_names + ['Target'])
    # Write data
    writer.writerows(train_data)

# Calculate the accuracy of the classifier on the training set
training_accuracy = accuracy_score(y_train, y_train_pred)
print("Training Accuracy:", training_accuracy)

# Sample data for prediction
sample = np.array([[4.4, 2.9, 1.4, 0.2]])

# Predict the class for the sample
prediction = nb_classifier.predict(sample)
print('Prediction:', prediction)
